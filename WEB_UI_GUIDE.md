# AI 多模态助手 - Web 界面使用指南

## 🚀 快速开始

### 1. 启动服务器

```bash
npm start
```

### 2. 打开浏览器

在浏览器中访问：

```
http://localhost:3000
```

## 🎨 界面功能

### 主要功能按钮

| 按钮 | 功能 | 用途 |
|------|------|------|
| 💬 智能对话 | AI 智能理解需求 | 自动判断使用哪个工具 |
| 🌤️ 天气 | 查询天气 | 输入城市名称 |
| 📈 股票 | 查询股票 | 输入股票代码 |
| 🎨 生成图片 | 生成图片 | 输入图片描述 |
| 🔊 语音播报 | 文字转语音 | 输入要播报的文字 |

### 快速示例

界面底部提供了一些快速示例按钮，点击后会在输入框中自动填入示例内容：

- **北京天气** - 查询北京天气
- **苹果股票** - 查询 AAPL 股票
- **上海图片** - 生成上海风景图片
- **语音播报** - 生成北京天气的语音
- **综合查询** - 同时查询天气和股票

## 📝 使用示例

### 示例 1: 查询天气

1. 在输入框输入：`北京`
2. 点击 **🌤️ 天气** 按钮
3. 查看返回的天气信息

### 示例 2: 查询股票

1. 在输入框输入：`AAPL` 或 `苹果公司`
2. 点击 **📈 股票** 按钮
3. 查看股票信息

### 示例 3: 生成图片

1. 在输入框输入：`上海晴天，黄浦江畔`
2. 点击 **🎨 生成图片** 按钮
3. 查看生成的图片

### 示例 4: 语音播报

1. 在输入框输入：`北京今天晴朗，气温15-20度`
2. 点击 **🔊 语音播报** 按钮
3. 播放生成的音频

### 示例 5: 智能对话

1. 在输入框输入：`今天北京天气怎么样？另外招商银行的股票涨了吗？`
2. 点击 **💬 智能对话** 按钮
3. AI 会自动调用天气和股票工具，综合回复

## 🎯 使用技巧

### 1. 使用智能对话
AI 会自动判断你的意图，选择合适的工具：
- "北京天气怎么样？" → 调用天气工具
- "苹果股票" → 调用股票工具
- "生成一张猫的图片" → 调用图片生成工具
- "用语音播报这段话" → 调用语音合成工具

### 2. 组合查询
可以在一次对话中查询多个内容：
```
北京今天天气怎么样？另外招商银行的股票涨了吗？请帮我生成一张天气图片。
```

### 3. 对话历史
如果需要上下文，可以连续提问：
```
你: 北京今天天气怎么样？
AI: 北京今天晴朗...

你: 那明天呢？
AI: 明天多云...
```

### 4. 使用示例按钮
点击快速示例按钮，可以快速体验各种功能，然后修改内容进行个性化查询。

## 🔧 高级功能

### 自定义图片尺寸
虽然界面默认使用 512x512，但你也可以在输入中指定：
```
生成一张1024x1024的高清图片：北京晴天
```

### 多语言支持
语音播报支持中文、英文、日文：
- 输入中文内容会生成中文语音
- 可以在代码中修改 `voice` 参数支持其他语言

### 图片保存
生成的图片会在浏览器中显示，右键可以保存到本地。

## ⚠️ 注意事项

1. **服务器必须运行**
   - 确保服务器已启动：`npm start`
   - 服务地址：`http://localhost:3000`

2. **网络连接**
   - 图片生成需要网络连接
   - 语音合成需要网络连接
   - 如果网络慢，图片生成可能需要几秒钟

3. **API 配置**
   - 确保 `.env` 文件中配置了 `DASHSCOPE_API_KEY`
   - 没有配置也可以使用基本功能，但某些高级功能可能受限

4. **浏览器兼容性**
   - 推荐使用 Chrome、Firefox、Safari 等现代浏览器
   - IE 不支持

## 🐛 故障排除

### 问题: 页面无法打开
- 检查服务器是否启动
- 检查端口 3000 是否被占用
- 尝试访问 `http://localhost:3000/health`

### 问题: 生成图片失败
- 检查网络连接
- 等待几秒钟再试
- 尝试更简单的描述

### 问题: 语音播报失败
- 检查网络连接
- 检查文字内容是否过长
- 确认浏览器支持音频播放

### 问题: 按钮没反应
- 等待上一次请求完成
- 查看浏览器控制台是否有错误
- 刷新页面重试
